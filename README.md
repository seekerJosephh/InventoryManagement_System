Question 1 – Active Directory Design and Deployment (33 marks)
Your university plans to centralize all computer and user management. Currently, each
department manages its own computers without a consistent policy. The university wants
centralized authentication, password complexity enforcement, shared folders for
departments, and the ability to disable accounts of graduated students.
Task:
Explain in detail how you would design and implement Active Directory Domain Services
(AD DS) for this university. Your answer must include:
1. Domain and Organizational Unit (OU) design
2. Domain controller installation steps
3. Account and Group management strategy
4. Group Policy implementation (password policy, login script, folder redirection, etc.)
5. Security considerations, backup strategy, and administrator roles
Your explanation must demonstrate your understanding of how AD DS solves real
administrative problems.
